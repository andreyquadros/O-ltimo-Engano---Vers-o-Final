# O Último Engano — Projeto Overleaf

**Como usar**
1. Crie um projeto no Overleaf e faça upload de todos os arquivos deste `.zip`.
2. Selecione **LuaLaTeX** como compilador (Menu > Settings > Compiler).
3. Edite os arquivos em `frontmatter/` e `chapters/` conforme for escrevendo.
4. Para KDP (6x9"), troque a geometria no `main.tex` (já tem comentário).
5. Boxes do framework **NIC** e **checklists** já prontos (tcolorbox).

**Estrutura**
- `frontmatter/` — pré-textuais (half-title, direitos, dedicatória, etc.)
- `chapters/` — capítulos por arquivo (inclui Prólogo e Epílogo)
- `appendix/` — apêndices, notas, bibliografia, sobre o autor, colofão
- `images/` — pasta para figuras (opcional)

Bom trabalho! ✍️
